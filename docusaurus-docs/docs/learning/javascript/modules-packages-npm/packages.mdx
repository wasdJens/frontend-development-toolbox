# Packages

- Packages sind Code Teile, welcher verteilt werden können
- Packages können ein oder mehrere Module beinhalten
- Packages können Abhängigkeiten zu anderen haben
- Für die Verwaltung von Packages benötigt man einen Paket Manager

# Node Package Manager

- NPM ist ein paket manager für Javascript
- NPM wird zusammen mit NodeJS installiert und kann als Programm ausgeführt werden
- NPM kann Packages installieren, hochladen und verwalten
