# Building and Bundling

- Javascript kann in unterschiedlichen Standards geschrieben werden (ECMA Script Versionen)
- Es gibt Supersets an Programmiersprachen die am Ende als normales Javascript ausgeführt werden (vgl. Typescript)
- Nicht alle Browser unterstützen jede Javascript Funktionalität
- In Javascript ist es üblich einen Bundler einzusetzen
- Beim Bau von einem Projekt wird der bestehende Code in ein Ziel Set von Javascript Compiled
 - Bspw: ECMA Script 2021 zu ECMA Script 2015
- Alle Module und Dateien werden zusammen gepackt in eine einzelne Datei (Bundling)
  - Angenommen wir hätten ca 200. Dateien. Diese über den Browser zu serven würde sehr viel Zeit beanspruchen
Stattdessen werden alle Dateien „gebündelt“ in eine einzelne Datei die dann später ausgeliefert wird.
