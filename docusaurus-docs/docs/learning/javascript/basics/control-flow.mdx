## Binary Operators

- Binary Operatoren für das vergleichen von Werten (Oder Variablen)

```js
console.log(3 > 2); // true
console.log(3 < 2); // false

console.log(25 >= 10); // true
console.log(100 <= 50); // false

console.log("Apple" == "Orange") // false
console.log("Apple" != "Cherry") // true
```

- Logische Operatoren

```js
console.log(true && false); // false
console.log(true && true); // true

console.log(true || false); // true
console.log(false || false) // false

// unary operator
console.log(!true); // false
console.loog(!false); // true
```

## Werte vergleichen

- `==` Abstract Equality (Datentyp Umwandlung)
- `===` Strict Equality (Keine Umwandlung)
- `Object.is` (Keine Umwandlung ES2015)
- Für das Vergleichen von Zwei Variablen sollte immer `===` verwendet werden
- Object.is sollte auch nur in Ausnahmen verwendet werden
  - Wenn man mit Float Numbers arbeitet und +0 und -0 unterschiedlich behandelt
  - Wenn man mit NaN arbeitet bei Floating Point Zahlen (Negative Unendlichkeit addiert zu positiver Unendlichkeit)

## Ternary Operator (?)

- Nimmt eine Bedingung und wertet diese aus
- Ist die Bedingung true wird der left hand operator ausgeführt
- Ist die Bedingung false wird der right hand operator ausgeführt

```js
console.log(true ? 1 : 2); // 1
console.log(false ? 1 : 2); // 2
```

## If Statement

- Bedinungen prüft man in Javascript mit `if`
- `if(condition)` evaluiert einen Ausdruck und sprint in den nächsten Block, wenn die Bedingung zutrifft (true)

```js
const price = 800;
const discount = 200;
let canPurchase;

// When price is equal or lower than 600 we want to purchase the item
if ((price - discount) <= 600) {
  canPurchase = true;
} else { // When price is larger than 600 we can not purchase it
  canPurchase = false;
}
console.log(canPurchase); // true
```

- Die zu prüfende Bedingung wird immer in einen Boolean Wert umgewandelt
- `0`, `null`, `undefined`, `NaN`, `''` (empty string) sind immer `false`

```js
let userInput =  null;
let hasUserInput;

if (userInput) {
  hasUserInput = true;
} else {
  hasUserInput = false;
}
console.log(hasUserInput) // false
```

- Verzweigungen können mit else und else if ausgeführt werden
- if kann auch ohne Verzweigung verwendet werden

```js
const price = 2000;
if (price < 2000) {
  console.log('Cheap!');
} else if (price > 2000) {
  console.log('Expensive!');
} else {
  console.log('Just right!');
}
```

## Switch Statement

- Mehrere if Statements können vereinfacht werden mit einem switch
- Switch hat einen oder n case Abschnitt und einen optionalen default Abschnitt
- Case Abschnitte werden mit break beendet

```js
let price = 2000;
switch (price) {
  case 2000:
    console.log('Just right!');
    break;
  case 1000:
    console.log('Cheap!');
    break;
  case 3000:
    console.log('Expensive!');
    break;
  default:
    console.log('Unknown price!');
}
```

## Loops

- Schleifen können Ausdrücke wiederholt ausführen
- In Javascript gibt es die gängigen Schleifen Konstrukte
  - for
  - while
  - do while
- Zusätzlich
  - for in
  - for of
- Mit break kann man einen Schleifen durchlauf abbrechen
- Mit continue kann man zur nächsten Iteration springen

### For Loop

- For loops wiederholen sich so lange bis eine Bedingung nicht mehr erfüllt ist
- Die JS for loop ist ähnlich zu Java und C

```js
for (begin; condition; step) {
  // repeat until false
}

for (let i = 0; i <= 100; i++) {
  console.log(i);
}
```

### Do ... While

- do .. while wiederholen sich bis eine Bedingung erfüllt ist
- Die Bedingung wird erst nach dem ersten Durchlauf geprüft

```js
do
  statement
while (condition);

let i = 0;
do {
  console.log(i)
  i++
} while (i <= 100);
```

### While

- while wiederholt sich solange die Bedingung erfüllt ist
- Die Bedingung wird vor dem ersten Durchlauf geprüft

```js
while(condition) {

}

let i = 0;
while (i <= 100) {
  console.log(i);
  i++;
}
```

### For ... in

- Die for … in Loop kann über Objekt properties loopen 

```js
const user = {
  name: "Jens",
  age: 25,
  gender: "m",
}

for (const i in user) {
  console.log(i); // name, age, gender
}
```

### For ... of

- Die for … of Loop iteriert über iterierbare Objekte (String, Array, Map, Set etc.

```js
const array1= ['A', 'B', 'C', 'D'];

for (const char of array1) {
  console.log(char); // A, B, C, D
}
```

## Kommentare

- In JS entweder mit // für single-line-comments
- Alternativ /* */ für multi-block-comments

```js
// This is a single line comment

/**
 * This is a multi-line comment
 */
```
