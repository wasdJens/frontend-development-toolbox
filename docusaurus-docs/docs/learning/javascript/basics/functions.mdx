## Grundlagen

- Funktionen werden mit dem `function` keyword deklariert
- An Funktionen können beliebige Parameter übergeben werden
- Funktionen sind Objekte in Javascript

```js
function square(number) {
  return number * number;
}

console.log(square(5)); // 25
```

- Funktionen geben immer `undefined` zurück
- Um einen Wert zurückzugeben verwendet man das `return` statement
- Primitive Paramter werden `pass by value` an die Funktion übergeben
- Nicht primitive Parameter werden `pass by reference` an die Funktion übergeben
- Das bedeutet wenn Sie eine Property verändern innerhalb der Funktion, wird diese Änderung auch außerhalb sichtbar

```js
const phone = {
  make: 'Apple',
  model: 'iPhone',
  year: '2021'
}

console.log(phone.year); // 2021

function updatePhone(phone) {
  phone.year = '2022';
}

updatePhone(phone);
console.log(phone.year); // 1995
```

## Function Expressions

- Funktionen können auch anonym deklariert werden
- Der Unterschied zu einer normalen Funktion ist der Funktion Name
- Können auch als `Immediately Invoked Function` Expression verwendet werden (Funktionen die sofort ausgeführt wird)

```js
const square = function(number) {
  return number * number;
}

console.log(square(5)) // 25
```

## Parameter

- Nicht übergebene Parameter werden `undefined`
- Default Parameter können angegeben werden

```js
function addPrefix(input, prefix = 'id: ') {
  return `${prefix} ${input}`
}
console.log(addPrefix('Test')); // id: Test
console.log(addPrefix('iPhone', 'unreleased')) // unreleased iPhone
```

## Funktionen als Parameter

- Funktionen  können auch als Parameter übergeben werden

```js
const upperCase = function(value) {
  return (value.toUpperCase())
}

function transformUserInput(upperCaseFunction, input) {
  return upperCaseFunction(input);
}

console.log(transformUserInput(upperCase, 'Hallo Welt')); // HALLO WELT
```

## Arrow Functions

- Funktionen können auch als `arrow functions =>` deklariert werden
- `Arrow functions` haben einige Einschränkungen gegenüber normalen Funktionen
- Geben implizit einen Rückgabewert zurück

```js
const sum = (a,b) => a +b;
console.log(sum(25,10)); // 35
```

- Kein binding auf `this` operator
- Sollten nicht als Methoden in Objekten deklariert werden
- Können nicht als Konstruktor verwendet werden
